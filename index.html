Política de Privacidade

Última atualização: 28 de fevereiro de 2026

Aplicativo: Lanly

Esta Política de Privacidade descreve como o aplicativo Lanly coleta, usa e protege as informações dos usuários. Ao usar o Lanly, você concorda com os termos descritos neste documento.

1. Informações que coletamos

O Lanly coleta apenas as informações estritamente necessárias para o funcionamento do app:

  • Nome do dispositivo: definido pelo próprio usuário nas configurações do app, usado para identificação na rede local.
  • Áudio gravado: capturado apenas quando o usuário mantém o botão PTT pressionado. O áudio é transmitido diretamente ao destinatário via rede local e não é armazenado em servidores externos.
  • Endereço IP local: usado exclusivamente para descoberta e conexão entre dispositivos na mesma rede Wi-Fi. Não é transmitido para fora da rede local.

2. Como usamos as informações

  • Identificar o dispositivo na rede local via mDNS/NSD
  • Estabelecer conexão ponto a ponto (P2P) entre dispositivos
  • Transmitir mensagens de voz PTT entre contatos aceitos

Não coletamos, não armazenamos e não compartilhamos dados pessoais com terceiros, exceto conforme descrito na seção de publicidade abaixo.

3. Microfone

O Lanly solicita permissão de acesso ao microfone exclusivamente para gravar mensagens de voz PTT. O microfone é ativado somente enquanto o botão "Segure para falar" estiver pressionado. Nenhum áudio é gravado em segundo plano.

4. Publicidade (Google AdMob)

O Lanly exibe anúncios através do Google AdMob. O AdMob pode coletar e usar dados do dispositivo para exibir anúncios relevantes, conforme a Política de Privacidade do Google .

Os dados coletados pelo AdMob podem incluir:

  • Identificadores de dispositivo (Advertising ID)
  • Dados de uso do app para personalização de anúncios
  • Endereço IP aproximado para segmentação geográfica

Você pode optar por não receber anúncios personalizados nas configurações do seu dispositivo Android em Configurações → Google → Anúncios.

5. Rede local

O Lanly opera exclusivamente em redes Wi-Fi locais. Toda a comunicação ocorre diretamente entre os dispositivos na mesma rede, sem passar por servidores externos. Nenhum dado de áudio ou mensagem é enviado para a internet.

6. Armazenamento de dados

O app armazena localmente no dispositivo apenas:

  • Nome do dispositivo (definido pelo usuário)
  • Lista de contatos aceitos (IDs locais)
  • Preferência de intercom ativo/inativo

Esses dados não são sincronizados com nenhum servidor externo.

7. Crianças

O Lanly não é destinado a crianças menores de 13 anos e não coleta intencionalmente informações de menores. Se você acredita que uma criança forneceu informações pessoais ao app, entre em contato conosco.

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas através de uma atualização do aplicativo. Recomendamos revisar esta página periodicamente.

9. Versão gratuita e comercial

O Lanly é atualmente gratuito para uso doméstico. Futuramente será lançada uma versão comercial com recursos adicionais para uso empresarial, que poderá ser paga. Os termos desta política se aplicam a ambas as versões.
